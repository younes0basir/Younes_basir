# Younes Basir Portfolio

A modern, professional portfolio website showcasing my work as a Software Engineering student and aspiring Data Analyst.

## 🌟 Features

- **Modern UI Design**: Clean, professional interface with glassmorphism effects and smooth animations
- **Dark Mode**: Full dark mode support with seamless theme switching
- **Responsive Design**: Fully responsive layout optimized for all screen sizes
- **Interactive Components**: 
  - Expandable project cards with detailed information
  - Animated hero section with floating elements
  - Tech stack visualization with progress bars
  - Contact form with mailto integration
- **Smooth Animations**: Reveal animations on scroll, hover effects, and transitions
- **Scroll Progress**: Visual progress indicator at the top of the page
- **Navigation**: Floating pill navbar that adapts on scroll

## 🚀 Tech Stack

- **Frontend**: React 18, Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/younes0basir/Younes_BASIR.git
```

2. Navigate to the project directory:
```bash
cd Younes_BASIR
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and visit `http://localhost:5173`

## 🛠️ Build for Production

To build the project for production:

```bash
npm run build
```

The optimized files will be in the `dist` directory.
